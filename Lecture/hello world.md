# Первый файл по контролю версий

## Выделение текста

Для выделения текста курсивом обраите его звездочками -
*Курсив*

Для выделения текста жирным обраите его двумя звездочками -
**Полужирны

Для зачеркивание текста обраите его двумя ~~ -
~~Зачеркнутый текст~~

## Списки

Для ненумерованого списка надо перед каждым тестом поставить звездочку и пробел
* Элемент 1
* Элемент 2
* Элемент 3

1. Первый элемент нумерованного списка.
2. Второй элемент нумерованного списка.
