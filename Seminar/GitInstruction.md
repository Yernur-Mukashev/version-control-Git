# Работа с Git
## 1. Проверка наличия установленного Git
В терминале выполнить команду `git --version`. Если Git установлен появится сообщение с ниформацией о версии программы, иначе будет сообщение об ошибке.

## 2. Установка Git
Загружем последнюю версию программы с сайта [Git](https://git-scm.com/downloads). Устанавливаем с настройками по умолчанию.
